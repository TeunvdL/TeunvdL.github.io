VernaCopter

About the Project

VernaCopter is an innovative system for drone control using natural language. It translates task descriptions into Signal Temporal Logic (STL) for autonomous motion planning. This project leverages Large Language Models (LLMs) for seamless human-drone interaction.

Key Features

  • Natural language to STL conversion
  • Autonomous motion planning
  • Interactive drone control
