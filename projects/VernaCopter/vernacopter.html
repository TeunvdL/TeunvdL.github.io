VernaCopter

Project Overview

VernaCopter is an innovative drone control system that leverages natural language processing and Signal Temporal Logic (STL) for motion planning. The goal of this project is to allow users to control drones by simply describing tasks in natural language, which are then converted into executable motion plans using STL. This breakthrough approach aims to simplify drone operation, making it more accessible to non-experts while still maintaining precision and efficiency.

Technical Details

At the heart of VernaCopter is a sophisticated motion planning system that combines natural language processing with signal temporal logic. The system takes input from users in the form of natural language descriptions, processes it using a pre-trained language model, and translates these instructions into executable motion plans. The motion plans are then executed by the droneâ€™s onboard control system, ensuring smooth and accurate flight paths.

  • Natural Language Processing (NLP): Using NLP techniques to process task descriptions.
  • Signal Temporal Logic (STL): Translating natural language into formal logic for motion planning.
  • Drone Control System: Integration with drone flight controllers for real-time execution of plans.
